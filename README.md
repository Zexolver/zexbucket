### Scoop Bucket (aka 'zexbucket')
A bucket I made to either:
- A) Make software installable through scoop that was not already installable through scoop, or
- B) Update software from other community scoop buckets that were outdated

The current pieces of software available to install through this bucket are:

- Friday Night Funkin'
- Ox, terminal text editor
- Shiru, anime watcher
